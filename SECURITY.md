# Security

This repository is temporarily public during bootstrap. Treat all committed content as public.

Do not open public issues, pull requests, discussions, or comments containing:

- secrets, tokens, passwords, private keys, or credentials;
- production URLs with embedded credentials;
- customer, tenant, or personal data;
- private architecture evidence;
- exploit packets or unrestricted red-team details;
- confidential business, legal, or commercial information.

For sensitive security matters, contact the repository owner through a private written channel. If GitHub private vulnerability reporting is enabled for this repository, use that path instead of public issues.

No production deployment, live-provider activation, public release, package publishing, or real-money activity is authorized from this repository during bootstrap.
