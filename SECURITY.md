# Security

## Supported Status

HADAF is in bootstrap and dogfood development. Public production activation,
GitHub Pages publication, package publication, live provider activation,
production credentials, and real-money activity are not authorized.

## Reporting

Report security concerns privately to the repository owner. Do not publish
exploit details, secrets, private prompts, credentials, tenant data, or restricted
evidence in public issues, pull requests, comments, logs, or artifacts.

## Repository Hygiene

This product repository must not contain Control Plane, Evidence Plane, Release
Plane, or Runtime Plane records. It must not contain secrets, private data,
private prompts, operational agent instructions, hidden telemetry, or local
machine paths.
