# HADAF

HADAF expands to HAUTM Agentic Delivery & Assurance Framework.

HADAF is a local-first, frontend-visible delivery and assurance system for turning approved plans into bounded implementation work, evidence, audits, assurance, and release handover.

This repository contains the clean HADAF Product Plane only. It must not contain private delivery records, local runtime state, local paths, secrets, credentials, or production access material.
